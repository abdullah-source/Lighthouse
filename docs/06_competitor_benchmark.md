## Strategic read

1. **The "action wedge" is closed.** AirOps' entire pitch is Insights → Action →
   Measure with an agent (Quill) that drafts and executes under approval gates and
   measures lift. Profound has agents + CMS publish. The "we act, they don't"
   positioning is dead, and so is "GitHub for marketing / one-click push" as a
   differentiator — that's Quill + Playbooks, shipping now.

2. **Measurement is commoditizing.** All three probe the same models and show
   near-identical Visibility / Position / Sentiment dashboards. Peec is explicitly
   winning on *simplicity* against the "feature overload" of the others. When the
   dashboard is a commodity, value moves to the causal layer.

3. **Nobody proves causal lift, and nobody owns consumer/DTC.** Public proof is
   generic ("headings 2.8x," "freshness 3x"). No category-dense causal dataset
   exists. And every visible customer is B2B SaaS — mid-market consumer/DTC
   (footwear, beauty, food) is an unclaimed lane (Profound has a "Shopping" SKU, so
   it's seen, not owned).

4. **Neutrality is marketed by no one** — confirmed. It's not a wedge as copy. It
   only has weight as a structural/ethical stance for a layer that shapes consumer
   recommendations.

## Implication for us

Differentiate on **proven causal lift, deep in one consumer category** — the dual-loop
(fast retrieval / slow training) framing as the intellectual spine, neutrality as the
earned ethical posture. Compete on a dataset that compounds, not on features they can
clone. See `01_friday_pitch_green.md` and `07_demo_experiment_spec.md`.

## GTM benchmarks our demo must clear

- Lead with the model logos (everyone does).
- Show a dashboard with Visibility / Position / Sentiment + competitor bars + trend.
- Hard proof numbers are the currency ("5x", "+79%", "24→68"). Bring a real number.
- A free audit/report is the universal entry wedge.
- "Act on insights" with specific source-type recommendations (G2, Reddit, PR) is
  table stakes — Peec shows it free on the homepage.
